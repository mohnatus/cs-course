# Десятичные числа

Число 678

## Дополнение до 9

```
678 + x = 999
```

Вычитаем в столбик:

```
    999
   -678
   ----
x = 321
```

## Дополнение до 10

```
678 + y = 1000
```

Вычитаем в столбик:

```
    1000
   - 678
   -----
y = ....
```

## Выводы

- Дополнение до 10 сложно вычислять: требуются заимствования из соседнего разряда (сложная логика)
- Для дополнения до 9 заимствования не требуются
- Дополнение до 10 = Дополнение до 9 + 1 (y = x + 1)

# Двоичные числа

Число 1010

## Дополнение до 1

Аналог дополнения до 9 для десятичных чисел.

```
1010 + x = 1111
x = 1111 - 1010
```

Вычитаем в столбик:

```
    1111
   -1010
   -----
x = 0101
```

### Инверсия

Дополнение до 1 - это то же самое, что инверсия числа:

для 1010 дополнение до 1 = 0101 = ~1010

## Дополнение до 2

Аналог дополнения до 10 для десятичных чисел.

```
1010 + y = 1_0000
y = 1_0000 - 1010
```

Вычитаем в столбик:

```
    10000
   - 1010
   ------
y = .....
```

Здесь также требуются заимствования - сложная логика вычисления.

Другой способ (через дополнение до 1):

Дополнение до 2 = Дополнение до 1 + 1:

y = ~1010 + 1 = 0101 + 1 = 0110

Проверка:

```
 ^^^
  1010 
+ 0110 
  ----
 10000
```


## Выводы

- Дополнение до 1 - это инверсия числа
- Дополнение до 2 = Дополнение до 1 + 1 = Инвертирсия + 1
